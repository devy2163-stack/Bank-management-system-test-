# 🏦 Bank Management System

A command-line Bank Management System developed in Python to demonstrate core Python programming concepts such as functions, file handling, JSON, loops, conditional statements, exception handling, and modular programming.

---

## 📌 Features

- 👤 Create New Account
- 💰 Deposit Money
- 💸 Withdraw Money
- 📊 Check Balance
- 📁 Store Data using JSON
- 🔒 Basic Input Validation

---

## 🛠 Technologies Used

- Python 3
- JSON

---

## 📂 Project Structure

```
Bank-Management-System/
│── app.py

│── hello.py
│── data.json
```

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/devy2163-stack/Bank-management-system-test-.git
```

2. Move inside the folder

```bash
cd Bank-management-system-test-
```

3. Run the project

```bash
python main.py
```
